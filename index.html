HABILIDADES

HTML

É a linguagem de marcação usada para criar e estruturar o conteúdo de uma página da web. Ele define os elementos que compõem a página, como parágrafos, cabeçalhos, links e imagens.

CSS

É uma linguagem de estilo usada para definir a apresentação e o layout de uma página da web escrita em HTML. Com CSS, você pode controlar aspectos visuais como cor, fonte, tamanho e posicionamento de elementos HTML.

C

É uma linguagem de programação de propósito geral amplamente usada. Criada nos anos 70, ela é conhecida por sua eficiência e portabilidade, sendo utilizada em uma variedade de aplicações, desde sistemas operacionais até software de aplicativos. É uma linguagem de programação de baixo nível, o que significa que oferece um alto nível de controle sobre o hardware do computador, mas requer mais detalhes em seu código.

JavaScript

É uma linguagem de programação de alto nível que adiciona interatividade e dinamismo às páginas da web. JavaScript é frequentemente usado para criar funcionalidades como validação de formulários, animações, manipulação do DOM e interações com o usuário.
